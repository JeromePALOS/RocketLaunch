# RocketLauch

Système d'allumage à distance de micro-fusée 

![Présentation](https://github.com/JeromePALOS/RocketLaunch/blob/main/asset/img/RocketLaunch.PNG?raw=true "Présentation")


## Instalation
- sur Raspberry pi 
- nodejs
- RaspAP pour connexion hostpost wifi

```
node index.js
```


## Defaut information
- link : localhost:8200
- Launch code : 1607
- Ouput Gpio : 4
