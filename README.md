# RocketLauch

Système d'allumage à distance de micro-fusée 

![Présentation](https://github.com/JeromePALOS/RocketLaunch/asset/img/RocketLaunch.PNG "Présentation")


## Instalation
- sur Raspberry pi 
- nodejs
- RaspAP pour connexion hostpost wifi

```
node index.js
```
